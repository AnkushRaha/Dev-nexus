# **TELEGRAM BOT**
This is a tlegram bot named DOOBBY who cracks jokes. I built it using python and joke api. This bot is very simple and basic. I think it is a really good beginner friendly project.

## **COMMANDS**
User can see the commands from telegram still i mention them here. 
 •/start - starts the bot.
 •/help - tells the user what this bot can do 
 •/joke - cracks a joke
