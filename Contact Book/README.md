# **Contact Book**------
Ever wanted to have a contact manager in your CLI ? That works exactly like the real one you use in you mobile. SO thats why this is a contact book application made with python that works fine on any CLI. 

## **Features**------
1. It works almost like the real contact manager but in CLI.
2. It saves the contacts in your storage.
3. you can easily search for a contact you want, delete one or rename.
